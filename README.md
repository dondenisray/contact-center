This project simulates a basic operational Business Intelligence use case
for a contact center.

## Stack
- SQL Server (data source)
- Python (pandas) for data quality validation
- Power BI for reporting

## KPIs
- Total Calls
- Answered Calls
- Answer Rate
- Average Handle Time (AHT)

## Focus
- Operational reporting
- Data quality and validation
- Trustable information for business users

This project is not focused on data science or machine learning.
